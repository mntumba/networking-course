# Practical Lab 3: Routing Configuration

## Overview
This practical lab focuses on various aspects of routing configuration, including static routing, router configuration, testing and verification, and troubleshooting scenarios.

## Static Routing
### Objectives:
- Understand the concept of static routing.
- Configure static routes on routers to direct traffic properly.

### Exercises:
1. Configure static routes between two routers.
2. Verify static routes using appropriate commands.

## Router Configuration
### Objectives:
- Gain hands-on experience in configuring routers for different scenarios.

### Exercises:
1. Set up basic router configurations (hostname, interfaces, IP addressing).
2. Implement and configure routing protocols (e.g., RIP, OSPF).

## Testing and Verification
### Objectives:
- Learn how to test and verify router configurations and routes.

### Exercises:
1. Use ping and traceroute commands to test connectivity.
2. Check routing tables and use debugging commands to verify routing operation.

## Troubleshooting Scenarios
### Objectives:
- Develop troubleshooting skills for common routing issues.

### Exercises:
1. Given a failed routing setup, diagnose the problem and propose solutions.
2. Perform troubleshooting using show commands and verify outputs.
