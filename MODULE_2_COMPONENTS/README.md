# Computer Components Study

This document provides an overview of various computer components and their functionalities. Understanding these components is essential for anyone interested in computer hardware.

## Main Components

1. **Central Processing Unit (CPU)**: The brain of the computer that performs calculations and processes data.
2. **Random Access Memory (RAM)**: Temporary storage that allows for quick access to data being actively used.
3. **Motherboard**: The main circuit board that connects all computer components.
4. **Storage**: Hard drives (HDD) and solid-state drives (SSD) used to store data permanently.
5. **Power Supply Unit (PSU)**: Converts electrical power from an outlet into usable power for the internal components of the computer.
6. **Graphics Processing Unit (GPU)**: Handles rendering of images and video, crucial for gaming and graphic design.

## Conclusion

Studying these components helps in understanding how computers operate and how to troubleshoot hardware issues.
