# Glossary of Networking Terms

## A
- **Address Resolution Protocol (ARP)**: A protocol used to map IP addresses to physical machine addresses in a local area network.
- **Asynchronous Transfer Mode (ATM)**: A networking technology that transfers data in packets or cells of a fixed size.

## B
- **Bandwidth**: The maximum rate of data transfer across a network.
- **BlueTooth**: A wireless technology standard for exchanging data over short distances.

## C
- **Client-Server Model**: A computing model that separates service providers (servers) from service requesters (clients).
- **Channel**: A medium used to transmit data.

## D
- **DNS (Domain Name System)**: The system that translates domain names into IP addresses.
- **DHCP (Dynamic Host Configuration Protocol)**: A network management protocol used to automatically assign IP addresses.

## E
- **Ethernet**: A system for connecting computers within a local area network.
- **Encapsulation**: The process of wrapping data with protocol information.

## F
- **Firewall**: A network security device that monitors and controls incoming and outgoing network traffic.
- **FTP (File Transfer Protocol)**: A standard network protocol for transferring files.

## G
- **Gateway**: A node that serves as a key stopping point for data on its way to or from other networks.
- **Gigabit Ethernet**: A version of Ethernet that supports data rates of 1 Gbps.

## H
- **HTTP (Hypertext Transfer Protocol)**: The foundation of data communication on the web.
- **HTTPS (HTTP Secure)**: An extension of HTTP that provides a secure communication over a computer network.

## I
- **IP Address**: A unique address for a device on a network that uses the Internet Protocol for communication.
- **ISP (Internet Service Provider)**: A company that provides individuals and organizations access to the Internet.

## J
- **Jitter**: The variability in time delay in packet delivery.
- **Java**: A high-level programming language widely used for building enterprise-scale applications.

## K
- **Kbps (Kilobits per second)**: A unit of data transfer rate
- **Key Exchange**: A method in cryptography to establish a secure key for communication.

## L
- **Latency**: The time it takes for data to travel from the sender to the receiver.
- **LAN (Local Area Network)**: A computer network that interconnects devices within a limited area.

## M
- **Modem**: A device that modulates and demodulates signals for data transmission.
- **Malware**: Software designed to disrupt, damage, or gain unauthorized access to a computer system.

## N
- **Network Topology**: The arrangement of different elements (links, nodes, etc.) in a computer network.
- **NAT (Network Address Translation)**: A method of remapping an IP address space into another IP address space.

## O
- **OSI Model (Open Systems Interconnection Model)**: A conceptual framework to understand and implement network protocols.
- **Outage**: A period when a service is unavailable.

## P
- **Proxy Server**: An intermediary server that separates end users from the websites they browse.
- **Packet**: A unit of data formatted for internet transfer.

## Q
- **QoS (Quality of Service)**: The overall performance of a network service.
- **Queuing**: The process of storing data packets until they can be processed.

## R
- **Router**: A device that forwards data packets between computer networks.
- **RDP (Remote Desktop Protocol)**: A protocol that allows a user to connect to another computer over a network connection.

## S
- **Subnetwork (Subnet)**: A segmented piece of a larger network.
- **SSL (Secure Socket Layer)**: A standard technology for keeping internet connections secure.

## T
- **TCP (Transmission Control Protocol)**: A core protocol of the Internet Protocol Suite, providing reliable, ordered, and error-checked delivery of a stream of data.
- **Topology**: The layout pattern of interconnections between devices in a network.

## U
- **UDP (User Datagram Protocol)**: A communication protocol that serves as a connectionless counterpart to TCP.
- **URL (Uniform Resource Locator)**: The address of a resource on the internet.

## V
- **VPN (Virtual Private Network)**: A service that encrypts your internet traffic and hides your IP address.
- **VoIP (Voice over Internet Protocol)**: A technology that allows you to make voice calls using the internet instead of a traditional phone line.

## W
- **WAN (Wide Area Network)**: A telecommunications network that extends over a large geographical area.
- **Wi-Fi**: A technology that allows electronic devices to connect to a wireless LAN.

## X
- **XSS (Cross-Site Scripting)**: A security vulnerability that allows an attacker to insert malicious scripts into a trusted website.

## Y
- **Yottabyte**: One of the largest units of data storage, equivalent to one trillion gigabytes.

## Z
- **Zigbee**: A specification for a suite of high-level communication protocols using low-power digital radios for personal area networks.
