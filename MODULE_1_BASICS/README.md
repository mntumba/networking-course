# Module 1: Hardware Fundamentals

## 1. Central Processing Unit (CPU)
A CPU, often referred to as the brain of the computer, executes instructions and performs calculations to carry out the tasks of modern computing.

### Specifications:
- **Cores:** More cores can improve multitasking and performance.
- **Clock Speed:** Measured in GHz, higher speeds yield better performance.
- **Cache Size:** Larger cache sizes enhance data retrieval speeds.

### Examples:
- Intel Core i9 (10th Gen): 10 cores, 3.7 GHz, 20MB cache.
- AMD Ryzen 5: 6 cores, 4.6 GHz, 16MB cache.

## 2. Random Access Memory (RAM)
RAM is a type of volatile memory that enables quick data access during computing processes.

### Types:
- **DDR3:** Older, slower, increasingly uncommon.
- **DDR4:** Standard in most modern PCs, faster transfer rates.
- **DDR5:** The latest version, offering even higher speeds and efficiency.

### Configurations:
- **Dual-channel:** Increases performance by using two RAM sticks concurrently.
- **Capacity:** Ranges from 4GB to 64GB or more in consumer-grade systems.

## 3. Storage Devices Comparison
Storage devices are essential for retaining data and applications.

| Device Type  | Speed    | Capacity        | Durability |
|--------------|----------|-----------------|------------|
| HDD          | Slower   | Up to 20TB      | Moderate   |
| SSD          | Fast     | Up to 8TB       | High       |
| NVMe         | Very Fast| Up to 16TB      | Very High  |

## 4. Motherboard Details
The motherboard connects all components of the computer.

### Key Features:
- **Chipset:** Determines compatibility and performance.
- **Form Factor:** ATX, Micro ATX, Mini ITX vary in size.
- **Expansion Slots:** PCIe slots for graphics cards and other peripherals.

## 5. Power Supply Specifications
A power supply unit (PSU) provides power to all computer components.

### Important Specs:
- **Wattage:** Must match system requirements (minimum 500W recommended for gaming).
- **Efficiency Rating:** 80 Plus certification levels (Bronze, Silver, Gold, Platinum).
- **Connectors:** Compatibility with motherboard and peripherals.

## 6. Cooling Systems
Cooling systems maintain optimal temperatures in a computer.

### Types:
- **Air Cooling:** Uses fans to dissipate heat; affordable and effective.
- **Liquid Cooling:** Uses liquid to remove heat; more efficient, quieter but potentially more expensive.

### Temperature Guidelines:
- **CPU Temp:** Ideally below 80°C under load.
- **GPU Temp:** Ideally below 85°C under load.

## 7. Input/Output Devices
I/O devices allow interaction with the computer.

- **Input Devices:** Keyboard, Mouse, Scanner.
- **Output Devices:** Monitor, Printer, Speakers.

## 8. Hardware Safety Precautions
Proper safety protocols in hardware handling are crucial.

- **ESD Protection:** Use anti-static wrist straps and mats.
- **Thermal Management:** Ensure adequate airflow and cooling solutions to prevent overheating.

## 10. Key Takeaways
- Understanding hardware fundamentals is critical for effective troubleshooting and upgrades.
- Regular maintenance, including dust cleaning and thermal paste application, prolongs component life.
