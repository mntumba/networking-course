## 1. Introduction to Networking
Networking is the practice of connecting computers and other devices to share resources. Networking enables communication between devices, often facilitating access to the internet, shared printers, and other services.

### Key Terms:
- **Node**: Any device connected to the network.
- **Packet**: A unit of data transmitted over a network.

## 2. OSI Model
The OSI (Open Systems Interconnection) model is a conceptual framework used to understand network interactions in seven layers. 

### Layers of the OSI Model:
1. **Physical Layer**: Hardware transmission.
2. **Data Link Layer**: Frame transmission.
3. **Network Layer**: Packet forwarding.
4. **Transport Layer**: End-to-end communication.
5. **Session Layer**: Session management.
6. **Presentation Layer**: Data format translation.
7. **Application Layer**: User interface.

### ASCII Diagram of OSI Model:
```
+---------------------------------+
|        Application Layer        |
+---------------------------------+
|       Presentation Layer        |
+---------------------------------+
|         Session Layer           |
+---------------------------------+
|        Transport Layer          |
+---------------------------------+
|         Network Layer           |
+---------------------------------+
|        Data Link Layer         |
+---------------------------------+
|         Physical Layer          |
+---------------------------------+
```

## 3. TCP/IP Model
The TCP/IP model is a concise framework for computer networking that includes four layers. It's essential for internet communication.

### Layers of TCP/IP Model:
1. **Link Layer**: Network interfaces.
2. **Internet Layer**: Packet routing.
3. **Transport Layer**: Data transmission.
4. **Application Layer**: Networking applications.

## 5. IP Addressing

## 6. Subnetting
Subnetting is the process of dividing a network into smaller, manageable sections.

### Benefits of Subnetting:
- Improved performance
- Enhanced security

## 7. Routing and Switching
Routing directs traffic between networks, while switching connects devices within the same network.

### Routing Protocols:
- **RIP**
- **OSPF**
- **BGP**
